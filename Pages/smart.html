Smart Cone Pencil Case

Project Introduction

The Smart Cone Pencil Case is a multi-functional device that helps students and office users maintain hygiene and utility in one smart package. Its core functionalities include:

  • Cleaning the pencils inside the case
  • Providing light for use in dark environments
  • Counting the number of pencils inside
  • Helping individuals with OCD by creating order and cleanliness
  • Reducing anxiety about using shared pencils

User Interaction

The Smart Cone Pencil Case supports multiple forms of interaction:

  • Voice Feedback: The device speaks the number of pencils currently inside.
  • Button Press: Users can press a button to activate the cleaning mechanism.
  • Light Activation: A separate button triggers the internal light for better visibility.
